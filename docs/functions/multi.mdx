# Multi-type

Explore available multi-type functions.

## chunk

Chunk an array or an object into small groups.

```js
const arr1 = [1, 2, 3, 4, 5];
const obj1 = { a: 1, b: 2, c: 3 };

ab.chunk(arr1, 3); // [[1, 2, 3], [4, 5]]
ab.chunk(obj1, 2); // [{ a: 1, b: 2 }, { c: 3 }]
```

```ts
export function chunk<T>(data: T, size: number): T;
```

| Parameter | Type                | Description                        |
|-----------|---------------------|------------------------------------|
| data      | `array` \| `object` | The data to chunk accordingly.     |
| size      | `number`            | The size chunked arrays will be.   |

## deduplicate

Remove duplicated elements in an array or an object.

```js
const arr1 = [1, 1, 5, 4, 4, 9, 8, 9];
const obj1 = { a: 1, b: 2, a: 4, b: 2 };

ab.deduplicate(arr1); // [1, 5, 4, 9, 8]
ab.deduplicate(obj1); // { b: 2, a: 4 }
```

```ts
export function deduplicate<T>(data: T): T;
```

| Parameter | Type                | Description                        |
|-----------|---------------------|------------------------------------|
| data      | `array` \| `object` | The data to remove duplicates.     |

---

## difference

Find the symmetric difference between an array, object or a string.

```js
ab.difference([1, 2, 3], [2, 3, 4]); // [1, 4]
ab.difference({ a: 1, b: 2, c: 3 }, { b: 2, c: 3, d: 4 }); // { a: 1, d: 4 }
ab.difference('abc', 'bcd'); // ['a', 'd']
```

```ts
export function difference<T, P>(a: T, b: P): T & P;
```

| Parameter | Type                             | Description                       |
|-----------|----------------------------------|-----------------------------------|
| a         | `array` \| `object` \| `string` | First data to compare.             |
| b         | `array` \| `object` \| `string` | Second data to compare.            |

## flatten

Unnest a nested array or object into a single-level structure.

```js
const arr1 = [1, [2, 3, 4], 5];
const obj1 = { a: 1, b: { c: 3, d: 4 } };

ab.flatten(arr1); // [1, 2, 3, 4, 5]
ab.flatten(obj1, '-'); // { a: 1, 'b-c': 3, 'b-d': 4 }
```

```ts
export function flatten<T>(data: T, joiner?: string): T;
```

:::info
Parameter `joiner` defaults to a period (a dot).
:::

| Parameter | Type                | Description                                                |
|-----------|---------------------|------------------------------------------------------------|
| data      | `array` \| `object` | The data to flatten.                                       |
| joiner?   | `string`            | String to join two object names when flattening an object. |

## intersection

Find duplicate elements in a object, string or an array.

```js
ab.intersection([1, 2, 3], [2, 3, 4]); // [2, 3]
ab.intersection({ a: 1, b: 2, c: 3 }, { b: 2, c: 3, d: 4 }); // { b: 2, c: 3 }
ab.intersection('abc', 'bcd'); // ['b', 'c']
```

```ts
export function intersection<T>(...datas: T[]): T;
```

| Parameter    | Type                            | Description                    |
|--------------|---------------------------------|--------------------------------|
| ...datas     | `array` \| `object` \| `string` | Datas to check duplicates.     |

---

## merge

Deeply merge an array, object or a string together.

```js
ab.merge([1, 2, 3], [4, 5, 6]); // [1, 2, 3, 4, 5, 6]
ab.merge({ a: 1, b: 2 }, { c: 3, d: 4 }); // { a: 1, b: 2, c: 3, d: 4 }
ab.merge('hello', 'hi'); // 'hellohi'
```

```ts
export function merge<T, P>(target: T, ...datas: P[]): T & P;
```

| Parameter | Type                            | Description                    |
|-----------|---------------------------------|--------------------------------|
| target    | `array` \| `object` \| `string` | Target to merge into.          |
| ...datas  | `array` \| `object` \| `string` | Datas to merge into target.    |

---

## reverse

Reverse an array, object or a string

```js
ab.reverse([1, 2, 3, 4]); // [4, 3, 2, 1]
ab.reverse({ a: 1, b: 2, c: 3 }); // { c: 3, b: 2, a: 1 }
ab.reverse('Hello, World!'); // '!dlroW ,olleH'
```

```ts
export function reverse<T>(data: T): T;
```

| Parameter | Type                            | Description                    |
|-----------|---------------------------------|--------------------------------|
| data      | `array` \| `object` \| `string` | The data to reverse.           |

---

## shuffle

Randomly shuffle an array, object or a string.

```js
ab.shuffle([1, 2]); // [1, 2] or [2, 1], etc.
ab.shuffle({ a: 1, b: 2 }); // { a: 1, b: 2 } or { b: 2, a: 1 } etc.
ab.shuffle('ab') // 'ab' or 'ba' etc.
```

```ts
export function shuffle<T>(data: T): T;
```

| Parameter | Type                            | Description                    |
|-----------|---------------------------------|--------------------------------|
| data      | `array` \| `object` \| `string` | The data to shuffle.           |

---

## sort

Sort an array, object or a string alphabetically or ascendingly.

```js
ab.sort([2, 3, 1]); // 1, 2, 3
ab.sort({ b: 2, c: 3, a: 1 }); // { a: 1, b: 2, c: 3 }
ab.sort('Aloha!', true); // '!Aahlo'
```

```ts
export function sort<T>(data: T, caseSensitive?: boolean): T;
```

| Parameter      | Type                            | Description                         |
|----------------|---------------------------------|-------------------------------------|
| data           | `array` \| `object` \| `string` | The data to shuffle.                |
| caseSensitive? | `boolean`                       | Whether uppercase is prioritized.   |
