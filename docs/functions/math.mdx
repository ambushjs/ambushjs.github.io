# Math

Explore available math utilities.

## factorial

Find the factorial of a number.

```js
ab.factorial(5);
// => 120
ab.factorial(10);
// => 3628800
```

```ts
export function factorial(n: number): number;
```

| Parameter | Type                | Description                                                |
|-----------|---------------------|------------------------------------------------------------|
| n         | `number`            | The number to caclulate the factorial.                     |

---

## fibonacci

Find the nth, or generate an array of the Fibonacci sequence.

```js
ab.fibonacci(2);
// => 1
ab.fibonacci(2, true);
// => [0, 1, 1]
```

```ts
export function fibonacci(n: number, array: boolean): number | number[];
```

| Parameter | Type                | Description                                                |
|-----------|---------------------|------------------------------------------------------------|
| n         | `number`            | The number to caclulate the factorial.                     |
| array     | `boolean`           | Determines wether to generate an array.                    |

## mean

Find the mean (average) in an array with numbers.

```js
ab.mean([1, 7, 4, 3]);
// => 3.75
ab.mean([1, 23, 3]);
// => 9
```

```ts
export function mean(numbers: number[]): number;
```

| Parameter | Type                | Description                                      |
|-----------|---------------------|--------------------------------------------------|
| numbers   | `number[]`          | Array of numbers to calculate the mean.          |

## pascal

Generate an array of the Pascal's triangle with the specified length.

```js
ab.pascal(3);
// => [ [ 1 ], [ 1, 1 ], [ 1, 2, 1 ] ]
ab.pascal(1);
// => [ [ 1 ] ]
```

```ts
export function pascal(n: number): number[][];
```
