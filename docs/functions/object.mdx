# Objects

Explore available object utilities.

## colors

An object with ANSI escape codes.

:::danger
Remember to always put `ab.colors.reset` at the end.
:::

```js
console.log(`${ab.colors.red}Hello!${ab.colors.reset}`);
// Try it yourself!
```
